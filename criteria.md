## 채점 방식 요약

- **컴파일 실패** → 전체 0점 (단, 입출력 구문 작성 시 +3점 부분 점수 가능)
- **컴파일 성공** → 각 문제 점수 합산
- 문제별 이슈(기능 미구현, 잘못된 출력 등) 발생 시 해당 문제 배점에서 감점

## 컴파일 점수

| 항목 | 배점 |
|------|------|
| 컴파일 성공 | 10점 |
| 컴파일 실패 + 입출력 구문 존재 | 3점 |
| 컴파일 실패 + 입출력 구문 없음 | 0점 |
