# 과제 채점 기준

## 과제명

HW1 — C Programming Homework 1 (가천대 소프트웨어학과, 최아영 교수)

## 채점 기준 (총 100점)

> 채점 방식: 컴파일 안 되면 0점. 입/출력 구문만 있으면 +3점. 컴파일 되면 10점 만점에서 이슈 발생 시 감점.

---

### 공통 감점 기준

- 전체 컴파일 실패: 해당 문제 0점
- 입/출력 구문(cin/cout)만 존재할 경우: +3점
- 컴파일 성공 기본: 10점에서 아래 이슈마다 감점

---

### HW1-1 (10점)

- 문제: PPT에 명시되지 않음 (이미지로만 제시)
- 컴파일 성공: 10점
- 컴파일 실패: 0점
- 결과 정확성 이상 없으면 만점, 오류 발생 시 감점

---

### HW1-2 (10점)

- 문제: 5자리 이하 정수를 입력받아 숫자 9의 개수를 세는 프로그램 (교재 3.38)
- 정수 입력(cin): 2점
- 각 자리 분리 로직(나눗셈/나머지 연산): 3점
- 9인 자리 카운트 정확성: 3점
- 결과 출력(cout): 2점
- 컴파일 실패: 0점

---

### HW1-3 (10점)

- 문제: 5자리 정수 입력 후 회문(palindrome) 여부 판단
- 정수 입력(cin): 2점
- 나눗셈/나머지로 각 자리 분리: 3점
- 첫째 자리 ↔ 다섯째 자리, 둘째 ↔ 넷째 비교 로직: 3점
- 결과 출력(palindrome/not palindrome): 2점
- 컴파일 실패: 0점

---

### HW1-4 (10점)

- 문제: 1~100 사이 소수 목록 출력 (교재 4.12)
- 소수 판별 알고리즘 구현: 5점
- 1~100 범위 순환(반복문): 2점
- 소수 목록 출력: 3점
- 컴파일 실패: 0점

---

### HW1-5 (10점)

- 문제: 양의 실수 N과 양의 정수 P를 입력받아 N^P를 for 루프로 계산
- float/double 입력(N)과 정수 입력(P): 2점
- for 루프 사용 (pow() 함수 사용 시 감점): 4점
- 결과 정확성: 2점
- 결과 출력: 2점
- 컴파일 실패: 0점

---

### HW1-6 (10점)

- 문제: 총 경과 초(seconds)를 입력받아 시:분:초 형식으로 변환 출력
- 초 입력(cin): 2점
- 시간 계산(/ 3600): 2점
- 분 계산(% 3600 / 60): 3점
- 초 계산(% 60): 1점
- 시:분:초 형식 출력: 2점
- 컴파일 실패: 0점

---

### HW1-7 (10점)

- 문제: 일일 운전 비용 계산 프로그램 (a~e 입력값으로 계산)
- a~e 항목 입력 처리: 4점
- 비용 계산 공식 정확성: 4점
- 결과 출력: 2점
- 컴파일 실패: 0점
